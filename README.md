# Python Pong Clone

A pong clone written in python using pygame with AI assistance but not abused.

## Usage

1. Ensure **git** and **python** is installed, then `git clone` the repo to your local folder.

2. `pip install pygame` to install pygame.

3. Run **main.py** using `python3 main.py`.

4. Enjoy the boring experience.

## Disclaimer

This **does not** modify, touch, nor access user files other than it's own directory.
